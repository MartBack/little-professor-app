# Little Professor Math App

> Educational math practice app inspired by the classic Little Professor toy from the 1970s

## Overview

A cross-platform desktop application that helps students practice basic arithmetic (addition, subtraction, multiplication, division) through an interactive, game-like interface similar to the original Little Professor educational toy.

## Getting Started

### Prerequisites

- Node.js 18+ and npm
- Git

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd little-professor-app

# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```
